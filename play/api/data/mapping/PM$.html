play.api.data.mapping

PM

object PM

Play provides you a Map[String, Seq[String]] in request body for urlFormEncodedRequest. It's generally a lot more convenient to work on Map[Path, Seq[String]] for defining Rules This object contains methods used to convert Map[String, Seq[String]] <-> Map[Path, Seq[String]]
